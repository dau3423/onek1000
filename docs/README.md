## 한 줄 요약 (TL;DR)

- **스택**: Next.js 14 (App Router) + TS + Tailwind + 카카오맵 SDK + Supabase(PostGIS) + Upstash Redis + Firebase App Hosting
- **데이터**: Opinet 무료 API → Cron 1h → Supabase / Redis 캐시 → 클라이언트
- **줌별 표시**: 1~6 시도 평균 → 7~9 시군구 TOP10 → 10~11 화면영역 TOP30 → 12+ TOP100(가격 라벨)
- **알람**: GPS 1km 반경에서 지역 평균 -50원 이상 시 상단 배너
- **BM**: 무료(AdSense 배너 + 상세 인터스티셜 일1회) / 1000냥 플랜 ₩1,000/월 (광고 제거 + 추후 즐겨찾기/푸시)
- **MVP 기간**: 풀타임 1명 기준 약 2주
