# 1000냥 주유소 - 프로젝트 개요

## 1. 프로젝트 정의

**서비스명**: 1000냥 주유소
**한 줄 정의**: 전국 주유소 실시간 가격을 지도에 띄워, 가장 싼 주유소로 사용자를 유도하는 웹 서비스. 월 1,000원만 내면 광고 없이 깔끔하게.
**타깃 사용자**: 기름값에 민감한 한국 운전자 (직장인 출퇴근, 장거리 운행자, 영업/물류 운전자, 라이더)

### 네이밍 의도
"1000냥"은 두 가지 의미를 동시에 담는다.
1. **싸다** — 1,000냥 = "1,000원샵"의 친근하고 저렴한 이미지
2. **구독료** — 월 1,000원만 내면 광고가 사라지는 명확한 가격표

## 2. 핵심 가치 제안 (Why 1000냥)

기존 오피넷 앱은 가격 정보는 정확하지만 UX가 무겁고 "지금 내 근처에서 어디가 가장 싼가"를 한눈에 보기 어렵다. 1000냥 주유소는 다음 두 가지에만 집중한다.

1. **줌 레벨에 따라 화면에 보이는 영역의 최저가 TOP N을 즉시 표시** — 사용자가 줌을 조작하는 것만으로 자연스럽게 후보군이 줄어든다.
2. **GPS 반경 1km 내 최저가 알람** — 내비 화면 옆에 띄워두면 주행 중 자연스럽게 인식되는 1km 권역 알림.

여기에 BM도 단순하다.
- **무료 사용자**: 광고를 봐도 모든 핵심 기능 사용 가능
- **유료 구독(₩1,000/월)**: 광고 제거 + 추후 부가 기능(즐겨찾기, 가격 알림 푸시, 주유 이력)

## 3. 핵심 기능 (MVP 범위)

### 3.1 지도 기반 가격 표시
- 대한민국 전체 지도(카카오맵) 기반
- 줌 레벨별 마커 노출 정책
  - 전국(zoom 1~6): 시/도 평균가 클러스터
  - 광역(zoom 7~9): 시/군/구 단위 최저가 TOP 10
  - 지역(zoom 10~11): 화면 영역 내 최저가 TOP 30
  - 동네(zoom 12~14): 화면 영역 내 최저가 TOP 100 (가격 라벨 직접 표시)
- 유종 필터: 휘발유 / 고급휘발유 / 경유 / LPG / 등유

### 3.2 GPS 기반 1km 반경 알람
- 사용자 위치 권한 동의 후 현재 위치 표시
- 1km 반경 내 최저가 주유소 자동 검출
- 일정 가격 차이(기본값: 평균 대비 -50원/L) 이상일 때 화면 상단 배너 알람
- 알람 해제 / 무시 옵션

### 3.3 주유소 상세 정보
- 상호, 브랜드(SK/GS/현대오일뱅크/S-OIL/알뜰/자영)
- 유종별 현재가
- 주소 / 전화번호
- 셀프 여부, 세차장 / 편의점 / 정비소 부가서비스
- 카카오맵 길찾기 연동 버튼

### 3.4 광고 / 구독 (BM 핵심)
- **무료 플랜**
  - 지도 하단 배너 광고 (AdSense 또는 카카오 모먼트)
  - 주유소 상세 페이지 진입 시 인터스티셜 광고 (일 1회로 제한 — 사용자 피로도 관리)
  - 모든 핵심 기능은 광고 시청과 무관하게 사용 가능
- **1000냥 플랜 (월 ₩1,000)**
  - 모든 광고 제거
  - (향후) 즐겨찾기 주유소 무제한
  - (향후) 즐겨찾기 주유소 가격 변동 푸시 알림
  - (향후) 내 주유 이력 기록 및 월 평균 단가 통계
- **결제 수단**
  - 토스페이먼츠 정기결제 또는 카카오페이 정기결제 (월 자동결제)
  - 무료 체험 7일 제공

## 5. 비즈니스 모델 상세

### 5.1 매출 가정 (보수적)
- DAU 10,000명 가정 시
  - 무료 사용자 95% = 9,500명 → 광고 노출. eCPM ₩1,000 가정 시 일 ₩9,500 × 30 = **월 ₩285,000**
  - 유료 사용자 5% = 500명 → ₩1,000 × 500 = **월 ₩500,000**
  - 합계: 월 **₩785,000** (DAU 1만일 때)
- DAU 10만명 도달 시 동일 비율 가정 → 월 **₩7,850,000**

### 5.2 비용 가정
- 서버 (Vercel + Supabase 또는 Cloudflare): 월 ₩50,000~200,000
- Opinet 무료 API 사용 (별도 비용 없음)
- 카카오맵 SDK 무료 한도 내 사용
- 결제 수수료(PG): 매출의 ~3.3%

### 5.3 전환 유도 포인트
- 광고가 가장 거슬릴 시점에 "₩1,000으로 광고 없애기" CTA 노출
  - 인터스티셜 광고 직후
  - 가격 알람 받았을 때 (이 기능을 푸시로 받으려면 구독 필요 — 알파 단계 이후)

## 6. 법적/저작권 고려사항

- Opinet 콘텐츠는 한국석유공사 저작권. 데이터를 그대로 재배포하지 않고 "지도 위 시각화" 형태로만 노출하는 것은 일반적으로 허용되나, 출처 표기 필수.
- 화면 하단 또는 정보 페이지에 "데이터 제공: 한국석유공사 오피넷" 명시
- **수익화 시 사전에 별도 협의/허락 필요 (오피넷 공지 기준)** — 광고/유료 구독 BM이므로 오피넷에 제휴 문의 필수 ((052)216-2509, price@knoc.co.kr)
- 위치 정보 수집 시 개인정보처리방침 명시 필요
- 정기결제(자동결제) 운영을 위해 전자상거래법상 표시·공시 사항 준수
