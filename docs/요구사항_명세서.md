# 1000냥 주유소 — 요구사항 명세서 (SRS)

> 본 문서는 `docs/01~06` 설계 문서와 실제 구현(`app/`, `lib/`, `components/`)을 분석해

## 1. 개요

### 1.1 제품 정의
- **서비스명**: 1000냥 주유소
- **한 줄 정의**: 전국 주유소 실시간 가격을 지도에 시각화하고, GPS 1km 반경 최저가를 알려주는 모바일 우선 웹 서비스.
- **타깃**: 기름값에 민감한 한국 운전자(출퇴근 직장인, 장거리/영업/물류 운전자, 라이더).
- **수익 모델(BM)**:
  - 무료 플랜: 광고 포함, 모든 핵심 기능 사용 가능.
  - 1000냥 플랜(₩1,000/월): 광고 제거 + 즐겨찾기 동기화 + 가격 변동 푸시. 7일 무료 체험.

### 1.2 데이터 출처
- 한국석유공사 **오피넷(Opinet) 무료 API**. 화면 하단/정보 페이지에 출처 표기 필수.
- 상업적 이용 시 오피넷 사전 협의 필요(법적 요구사항, §8 참조).

### 1.3 용어
| 용어 | 의미 |
|------|------|
| 유종 코드 | B027(휘발유), B034(고급휘발유), D047(경유), K015(등유), C004(LPG) |
| bbox | 지도 화면 영역(SW/NE 좌표)으로 조회하는 범위 |
| radius | GPS 현재 위치 기준 반경 조회 |
| Mock 모드 | `NEXT_PUBLIC_USE_MOCK=true`일 때 외부 의존 없이 동작하는 개발 모드 |
| 프리미엄/구독자 | 1000냥 플랜 결제(또는 무료 체험) 활성 사용자 |

## 3. 기능 요구사항 (FR)

각 항목은 `FR-x.y` ID, 우선순위(P0 필수 / P1 중요 / P2 향후), 구현 위치, 수용 기준(AC)을 가진다.

### 3.1 지도 기반 가격 표시 (P0)

- **FR-1.1** 카카오맵 기반 전국 지도 렌더. SSR 시 서울 시청 중심으로 초기화.
  - 위치: `components/map/KakaoMap.tsx`, `app/page.tsx`
- **FR-1.2** 줌 레벨별 마커 노출 정책
  - zoom 1~6: 시/도 평균가 클러스터
  - zoom 7~9: 시/군/구 단위 최저가 TOP 10
  - zoom 10~11: 화면 영역 내 최저가 TOP 30
  - zoom 12~14: 화면 영역 내 최저가 TOP 100 (가격 라벨 CustomOverlay 직접 표시)
- **FR-1.3** 유종 필터(휘발유/고급/경유/LPG/등유) + 브랜드 + 셀프 필터. 선택 즉시 `bbox` 재조회.
  - 위치: `components/ui/FilterBar.tsx`, `stores/map.ts`
- **FR-1.4** 마커 색상 규칙: 화면 평균 대비 -30원↓ 초록 / ±30원 노랑 / +30원↑ 빨강(줌 12+).
- **AC**: 지도 패닝/줌 시 해당 영역의 마커가 정책에 맞게 갱신되고 60fps를 유지한다(클러스터링 활용).

### 3.2 GPS 1km 반경 알람 (P0)

- **FR-2.1** 위치 권한 동의 후 `navigator.geolocation.watchPosition`으로 현재 위치 추적.
  - 위치: `hooks/useGeolocation.ts`
- **FR-2.2** 100m 이상 의미있는 이동 시 `radius` API 재조회(좌표 100m 양자화).
- **FR-2.3** 1km 반경 최저가가 지역 평균 대비 **-50원/L 이상** 저렴하면 상단 배너 알람. 임계값은 설정에서 조정 가능.
  - 위치: `components/alert/RadiusAlert.tsx`
- **AC**: 위치 동의 후 더 싼 주유소 발견 시 상단 배너가 노출되고, 탭하면 해당 주유소로 이동한다.

### 3.3 주유소 상세 (P0)

- **FR-3.1** 상호/브랜드/주소/전화/셀프여부/부가서비스 표시.
- **FR-3.2** 유종별 현재가 + 마지막 보고일자 표시.
- **FR-3.3** 카카오맵 길찾기 외부 링크 + 전화걸기 CTA.
- **FR-3.4** 30일 가격 이력 차트(Recharts). 위치: `components/charts/PriceHistoryChart.tsx`, `app/api/stations/[id]/history/route.ts`
- **FR-3.5** (P1) 리뷰: 별점 1~5, 본문 500자, 사진 최대 5장. 위치: `components/reviews/*`, `app/api/stations/[id]/reviews/route.ts`

### 3.4 검색 / 경로 (P1)

- **FR-4.1** 주유소 상호/주소 검색(pg_trgm). 위치: `app/search/`, `app/api/search/route.ts`
- **FR-4.2** 경로별 최저가: 출발/도착 직선 buffer 2km PostGIS RPC. 위치: `app/route/`, `app/api/route-cheapest/route.ts`

### 3.5 인증 (P0)

- **FR-5.1** NextAuth 카카오/구글 OAuth 로그인. 위치: `app/api/auth/[...nextauth]/route.ts`, `lib/auth/options.ts`
- **FR-5.2** 세션에 `isPremium`/`subStatus` 주입(60초 캐시). 위치: `lib/auth/session.ts`, `types/next-auth.d.ts`

### 3.6 광고 / 구독 (P0)

- **FR-6.1** 무료 사용자: 지도 하단 sticky 배너 + 상세 진입 시 인터스티셜(일 1회, localStorage flag).
  - 위치: `components/ads/*`
- **FR-6.2** 광고 SDK 로딩 실패 시 페이지 영향 0 (ErrorBoundary/빈 div fallback). AdSense 미설정 시 자체 CTA 폴백.
- **FR-6.3** 구독자에게는 모든 광고 비노출(SSR 서버 검증 기반 분기).
- **FR-6.4** KG이니시스 결제. 사용자는 두 방식 중 선택:
  - 단건(1개월권): INIStdPay 표준결제창으로 ₩1,000 1회 결제 → 1개월 이용 후 만료(자동갱신 없음, 무료체험 없음).
  - 정기 구독: 빌링키(CARD_BillKey) 발급 → 7일 무료 체험 → 매월 자동결제(빌링 REST API). 카드 빌링 중심.
  - 결제창은 카드·계좌이체·간편결제(카카오페이/토스페이) 노출. 정기(빌링)는 카드 중심.
  - 위치: `app/api/billing/{subscribe,return,cancel,webhook,charge-cron}/route.ts`, `lib/billing/inicis.ts`
- **FR-6.5** 정기결제 Cron(매시). 단건 만료 처리 + 정기 빌링 청구·실패 재시도(3회 past_due)·해지 처리.

### 3.7 즐겨찾기 / 푸시 (P1)

- **FR-7.1** 즐겨찾기 추가/조회/동기화. 위치: `app/api/favorites/route.ts`, `components/FavoriteButton.tsx`
- **FR-7.2** 즐겨찾기 가격 변동 감지 → 프리미엄 사용자 Web Push. 위치: `lib/push/webpush.ts`, `public/sw.js`, sync-opinet Cron 내장.

### 3.8 데이터 동기화 (P0)

- **FR-8.1** Cron 매시 → `/api/internal/sync-opinet`: 시도×유종 Opinet 호출 → 변경분 `prices_latest` UPSERT → 푸시 발송.
  - 보호: `Authorization: Bearer ${CRON_SECRET}` 검증 필수.
- **FR-8.2** Opinet 클라이언트(xml/json 파서, 지역/유종 코드 상수). 위치: `lib/opinet/client.ts`

## 4. 데이터 모델 (요약)

- `stations`: 주유소 마스터(id=Opinet UNI_ID, brand, name, sido/sigungu, address, tel, is_self, 부가서비스, `geom geography(Point,4326)` + GiST 인덱스, lat/lng 컬럼 분리).
- `prices_latest`: (station_id, product) PK, price, trade_dt, updated_at. `(product, price)` 인덱스.
- `prices_history`: 가격 이력(차트/변동 감지용).
- `users` / `subscriptions`: 사용자/구독(빌링키, 상태, 다음 결제일).
- `favorites`: 사용자×주유소.
- `reviews`: 별점/본문/사진(`review-photos` 버킷).
- 마이그레이션: `supabase/migrations/0001~0004`.

## 7. 보안 요구사항

- **SEC-1** Opinet API 키는 서버 전용(`OPINET_API_KEY`, public 금지).
- **SEC-2** `/api/internal/*`, `/api/billing/charge-cron`은 `CRON_SECRET` Bearer 검증.
- **SEC-3** `/api/stations/*` IP당 분당 60회 rate limit(Upstash).
- **SEC-4** 카카오맵 JS 키는 도메인 허용목록 잠금.
- **SEC-5** 구독자 판정은 반드시 서버 검증(클라이언트 신뢰 금지).
- **SEC-6** KG이니시스 결제 검증: returnUrl 결과는 신뢰하지 않고 authToken/authUrl로 S2S 승인요청해 확정. authUrl 도메인(inicis.com) 검증. signKey/INIAPI Key는 서버 전용(NEXT_PUBLIC_ 금지, mid만 노출).
- **SEC-7** 업로드 사진: 타입(image/*)·크기(5MB) 검증, 비공개 버킷.

## 9. 코딩 컨벤션 (리뷰 기준)

> 실제 코드베이스에서 관찰된 규칙. 신규/수정 코드는 이를 따른다.

- **언어/주석**: 한국어 주석·UI 문자열 사용(기존 톤 유지). 코드 식별자는 영문.
- **TypeScript**: strict, `any` 지양. 도메인 타입은 `types/`에 정의(`Station`, `ProductCode` 등) 후 재사용.
- **컴포넌트**: 클라이언트 컴포넌트는 최상단 `'use client'`. 기본 export 함수형 컴포넌트.
- **스타일**: Tailwind 유틸리티 우선. 다크모드는 `dark:` variant. safe-area 고려.
- **상태**: 클라이언트 전역상태=Zustand(`stores/`), 서버상태=TanStack Query.
- **데이터 접근**: DB 쿼리는 `lib/db/queries.ts`에 모으고 route는 얇게 유지.
- **Mock 우선**: 모든 외부 의존(Opinet/Supabase/Redis/결제/푸시)은 `NEXT_PUBLIC_USE_MOCK`로 분기되어 키 없이 로컬 동작 가능해야 함.
- **이미지**: `next/image` 사용, width/height 명시.
- **검증 명령**: `npm run typecheck`, `npm run lint`, `npm run build` 통과 필수.

## 10. 완료/검증 기준 (Definition of Done)

신규 작업은 다음을 모두 만족해야 "완료"로 간주한다.

1. 해당 FR/NFR의 수용 기준(AC) 충족.
2. `npm run typecheck` + `npm run lint` 무오류.
3. Mock 모드(`NEXT_PUBLIC_USE_MOCK=true`)에서 외부 키 없이 동작 확인.
4. 보안 요구사항(§7) 위반 없음(특히 키 노출/인증 누락).
5. 모바일 뷰포트(iOS Safari/Android Chrome) 레이아웃 깨짐 없음, 다크모드 확인.
6. 기존 컨벤션(§9) 준수, 불필요한 의존성 추가 없음.
7. 변경 범위에 맞는 최소 단위 커밋 메시지(한국어, 기존 형식 `feat:`/`fix:`/`chore:`).
