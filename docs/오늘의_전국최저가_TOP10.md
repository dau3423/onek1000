## 0. 데이터 채우는 법 (필수)

- **소스**: 오피넷 `lowTop10.do` (전국 시도별 최저가를 모아 가격 오름차순 정렬 → 상위 10개)
- **이미 구현됨**: `lib/opinet/client.ts`의 `fetchAllStationsByRegion(product)` 가 17개 시도를 순회하며 `lowTop10.do`(cnt=20)를 호출해 풀을 만든다. 이 결과를 `price` 오름차순 정렬 후 상위 10개를 쓰면 된다.
- **유종 코드(prodcd)**: 휘발유 `B027`, 경유 `D047`
- **필드 매핑**: `OS_NM`→상호, `PRICE`→리터당 가격, `NEW_ADR`→주소(지역 추출), `POLL_DVS_CD`→브랜드, `isSelf`→셀프 여부
- **전국 평균**: `avgSidoPrice.do`(`fetchAvgBySido`)의 시도 평균을 단순평균하거나 오피넷 전국 평균값 사용
- 치환자 목록: `{{날짜}}`(예: 2026.6.7), `{{유종}}`(휘발유/경유), `{{1위_가격}}`…`{{10위_가격}}`, `{{N위_지역}}`, `{{N위_상호}}`, `{{N위_브랜드}}`, `{{전국평균}}`, `{{최대차액}}`(전국평균 − 1위가격)
