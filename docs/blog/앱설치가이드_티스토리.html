⛽ 1000냥 주유소 앱 설치 방법 (아이폰·안드로이드)

별도 앱스토어 설치 없이, 홈 화면에 추가 한 번이면 끝! 전국 주유소 최저가를 아이콘 하나로 바로 확인하세요.

👉 먼저 접속
브라우저 주소창에 onek1000.kr 을 입력해 접속하세요.

🍎 아이폰(iOS) 설치 — Safari

⚠️ 반드시 사파리(Safari)로 접속하세요. 크롬·카카오톡·인스타 인앱 브라우저에서는 '홈 화면에 추가'가 안 됩니다.

  1. Safari로 onek1000.kr 접속
  2. 화면 하단 가운데 공유 버튼(□ 위로 화살표 ⬆️) 탭
  3. 메뉴를 내려서 ‘홈 화면에 추가’ 탭
  4. 오른쪽 위 ‘추가’ 탭
  5. 홈 화면에 생긴 1000냥 주유소 아이콘으로 실행 🎉

🤖 안드로이드 설치 — Chrome

  1. Chrome(크롬)으로 onek1000.kr 접속
  2. 주소창 오른쪽 설치 아이콘 또는 오른쪽 위 메뉴(⋮) 탭
  3. ‘앱 설치’ 또는 ‘홈 화면에 추가’ 탭
  4. ‘설치’ 탭 → 홈 화면 아이콘으로 실행 🎉

⚠️ 카카오톡·인스타 등으로 받은 링크는 ‘인앱 브라우저’로 열립니다. 이때는 화면 안내의 ‘외부 브라우저로 열기’ 또는 메뉴의 ‘다른 브라우저로 열기’로 크롬/사파리에서 여세요.

✨ 설치하면 좋은 점

  • 홈 화면 아이콘으로 1초 실행, 전체 화면으로 쾌적하게
  • 가격 하락 알림: 즐겨찾기한 주유소가 싸지면 푸시로 알려줘요(알림 ‘허용’ 필요)
  • 설치 용량 부담 없이 항상 최신 가격 반영
