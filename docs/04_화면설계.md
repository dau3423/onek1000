# 1000냥 주유소 - 화면 설계 (와이어프레임)

## 1. 페이지 맵

```
/                       메인 지도 화면 (랜딩)
  ├─ #panel:filter      유종/브랜드 필터
  ├─ #panel:list        화면 내 TOP N 리스트 시트 (하단 sheet)
  └─ #alert:radius      1km 알람 (상단 배너)

/station/[id]           주유소 상세
  └─ #cta:directions    카카오맵 길찾기

/pricing                구독 안내 (1000냥 플랜)
/my                     마이페이지 (구독 관리, 즐겨찾기)
/auth/sign-in           로그인 (카카오/구글)
/legal/{terms|privacy}  약관/개인정보
```

### 주요 컴포넌트 동작
- **필터 바**: 유종(휘발유/고급/경유/LPG/등유) + 브랜드 + 셀프만. 선택 즉시 `bbox` API 재호출
- **알람 배너**: 1km 반경 최저가가 지역 평균 대비 -50원 이상일 때만 노출. 탭하면 해당 주유소로 지도 이동 + 상세 sheet open
- **마커 색상 규칙**
  - 🟢 초록: 화면 평균보다 -30원 이상 저렴
  - 🟡 노랑: 평균 ±30원
  - 🔴 빨강: 평균보다 +30원 비쌈 (줌 12+에서만 노출)
- **가격 라벨 (CustomOverlay)**
  - 줌 12 이상: "₩1,580" 텍스트 노출
  - 줌 7~11: 마커 핀만 표시, 탭 시 가격 툴팁
  - 줌 1~6: 시도 평균가 클러스터 (예: "서울 ₩1,623")
- **하단 시트**: 기본 peek(56px), 드래그업하면 풀 시트. 시트 아이템 탭 → 마커 강조 + 상세 페이지 진입

## 6. 광고 노출 정책 (무료 사용자)

| 위치 | 형태 | 빈도 |
|------|------|------|
| 메인 지도 하단 | 320×50 또는 320×100 배너 (sticky) | 상시 |
| 상세 페이지 진입 | 인터스티셜 (전면) | 일 1회 |
| 검색 결과 리스트 | 네이티브 광고 카드 | 5번째 항목마다 |

전환 유도(₩1,000 구독) CTA는 인터스티셜 닫기 직후 + 마이페이지 헤더에 비강제 노출.

## 7. 인터랙션 핵심 흐름

### 7.1 첫 방문 (게스트)
1. `/` 진입 → SSR로 서울 시청 중심 지도 + 시도 평균가 표시
2. "내 위치로 이동" 탭 → 위치 권한 요청
3. 동의 → `radius` API 호출 → 알람 조건 충족 시 배너 노출
4. 5분 사용 후 인터스티셜 광고 1회 + "₩1,000으로 광고 없애기" CTA

### 7.2 구독 전환
1. 광고/CTA 클릭 → `/pricing`
2. "1000냥으로 시작하기" → 카카오/구글 로그인 → 토스 빌링키 등록
3. 7일 무료 체험 시작 → 즉시 광고 비활성
4. 7일 후 자동 결제 → 영수증 메일

### 7.3 구독 해지
1. `/my` → "구독 관리/해지" → 해지 사유(선택) → 확인
2. 다음 결제일에 만료, 만료 전까지 광고 없음 유지

## 8. 디자인 시스템 가이드 (요약)

| 토큰 | 값 | 용도 |
|------|------|------|
| `--color-primary` | #FF6B00 | 1000냥 브랜드 (주황 — 주유소/저렴함 연상) |
| `--color-cheap` | #16A34A | 싼 가격 (초록) |
| `--color-normal` | #EAB308 | 평균 (노랑) |
| `--color-expensive` | #DC2626 | 비싼 가격 (빨강) |
| `--color-bg` | #FFFFFF / #0B0B0F (dark) | 배경 |
| `--font-display` | Pretendard | 한글 가독성 |
| `--radius-md` | 12px | 카드/시트 |
| `--shadow-sheet` | 0 -8px 24px rgba(0,0,0,.12) | 하단 시트 |

모바일 안전영역(safe-area-inset-*) 반드시 적용. 다크모드 별도 토큰 1차 확보.
