# 3. 사업 내용

이용자 단말기의 GPS 위치정보를 이용하여 **현재 위치 주변 주유소의 실시간 유종별(휘발유·고급휘발유·경유·LPG·실내등유) 최저가 정보를 지도 기반으로 제공**하는 서비스입니다.

주요 기능은 다음과 같습니다.

- **지도 기반 가격 표시**: 전국 주유소의 실시간 가격을 지도에 시각화하여 제공합니다.
- **내 위치 기반 주변 최저가**: 이용자가 위치 권한에 동의한 경우, 현재 위치(위·경도)를 이용해 일정 반경(예: 10km) 내 최저가 주유소 TOP 10을 검색·표시합니다.
- **반경 최저가 알림**: 현재 위치 반경 1km 내 최저가가 지역 평균 대비 일정 수준 이상 저렴할 경우 알림을 제공합니다.
- **경로별 최저가**: 출발지·도착지 경로상의 최저가 주유소를 제공합니다.
- **관심 지역 알림(선택)**: 이용자가 직접 등록한 관심 지역(집·회사 등 고정 좌표)에 대해 가격 변동 알림을 제공합니다.
- **주유소 상세·리뷰·즐겨찾기** 등 부가 기능.

가격 정보는 한국석유공사 오피넷(Opinet) 등 공개 데이터를 기반으로 수집·가공하여 제공합니다.

# 4. 위치정보의 수집·이용 항목 및 방식

| 항목 | 수집 시점 | 이용 목적 | 저장 여부 |
|------|-----------|-----------|-----------|
| 현재 위치(GPS 위·경도) | 이용자가 위치 권한에 동의하고 위치 기반 기능 사용 시 | 주변 최저가 주유소 검색·표시, 반경 알림 | **저장하지 않음(일회성 처리)** |
| 관심 지역 좌표(위·경도) | 이용자가 직접 관심 지역을 등록 시 | 해당 지역 가격 변동 알림 | 이용자 계정과 연계하여 저장 |

- 실시간 현재 위치는 서비스 제공을 위해 일시적으로만 이용하며, 별도로 저장하지 않습니다.
- 관심 지역 좌표는 이용자가 명시적으로 입력·등록한 고정 지점에 한해 저장되며, 이용자가 언제든 삭제할 수 있습니다.

# 5. 위치정보시스템의 구성 및 운영

서비스의 위치정보 처리 흐름은 다음과 같습니다.

1. 이용자 단말(웹 브라우저)에서 위치 권한 동의 후 GPS로 현재 위치를 획득합니다.
2. 획득한 위·경도 좌표를 HTTPS(TLS) 암호화 통신으로 서비스 서버에 전송합니다.
3. 서버는 위치정보 데이터베이스(PostGIS 기반)를 이용해 해당 좌표 반경 내 주유소와의 거리를 계산하고 최저가 순으로 정렬합니다.
4. 계산 결과(주변 최저가 주유소 목록)를 단말에 반환하여 지도에 표시합니다.

- 실시간 위치 좌표는 4단계 처리 후 저장 없이 폐기됩니다.
- 관심 지역 기능의 경우에만, 이용자가 등록한 좌표를 데이터베이스에 저장합니다.
- 서버 인프라: Google Firebase App Hosting(Cloud Run), 데이터베이스: Supabase(PostgreSQL/PostGIS).

# 6. 위치정보 보호를 위한 기술적·관리적 조치

- **전송 구간 암호화**: 단말과 서버 간 모든 통신은 HTTPS(TLS)로 암호화합니다.
- **최소 수집·목적 외 이용 금지**: 서비스 제공에 필요한 범위에서만 위치정보를 이용하며, 목적 외 이용을 금지합니다.
- **일회성 처리**: 실시간 현재 위치는 저장하지 않고 즉시 처리 후 폐기합니다.
- **접근 권한 관리 및 접근 기록 보관**: 데이터베이스 접근 권한을 최소화하고 접근 기록을 관리합니다.
- **동의 기반 수집·철회 보장**: 이용자 동의를 받은 경우에만 위치를 수집하며, 기기·브라우저 설정을 통해 언제든 권한을 철회할 수 있습니다.
- **고지**: 개인정보처리방침에 위치정보의 수집·이용·보관에 관한 사항을 고지합니다.
